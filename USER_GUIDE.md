# DCMS 用户指南

## 简介
DCMS 是一个面向 DOTA2 社区赛事的轻量管理系统，当前版本支持以下核心场景：
- 玩家库管理
- 数字 ID 登录
- 赛事创建与报名
- 内战队长任命与选人
- 选手拍卖与竞价

当前系统主要分为 3 个页面：
- `首页`
- `选手拍卖页`
- `内战选人页`

## 用户角色
系统当前主要有 3 类使用者：

### 1. 管理员
管理员负责：
- 创建赛事
- 管理玩家库
- 开启或关闭赛事报名
- 代玩家报名
- 任命队长并初始化赛事
- 创建拍卖并分配队长预算
- 启动、暂停、继续拍卖

### 2. 普通玩家
普通玩家可以：
- 通过数字 ID 登录
- 在首页查看当前赛事
- 报名或取消报名某个赛事
- 查看公开玩家库

### 3. 队长
队长是特殊玩家身份，在赛事中被管理员任命后可以：
- 在内战选人页进行选人
- 在拍卖页代表自己的队伍出价

## 首页使用说明

### 未登录状态
首页提供两种登录方式：
- 管理员登录
- 玩家数字 ID 登录

### 玩家登录后
玩家登录后在首页可以看到：
- 当前赛事列表
- 每个赛事的报名状态
- 已报名人数
- 当前赛事阶段
- 公开玩家库

玩家可执行操作：
- 点击 `报名` 参加某个开放报名的赛事
- 点击 `取消报名` 退出自己已报名的赛事
- 在公开玩家库中按关键词、位置、战力排序查看玩家资料

### 管理员登录后
管理员首页会展示：
- 玩家总数
- 赛事总数
- 报名开放中的赛事数量
- 当前拍卖状态

管理员首页同时包含 3 个管理区：
- 玩家库管理
- 赛事与报名管理
- 拍卖配置管理

## 玩家库管理
管理员可在 `玩家库管理` 中完成以下操作：
- 新增玩家
- 编辑玩家
- 删除未参与赛事流程的玩家
- 按昵称、数字 ID、微信昵称搜索玩家
- 按位置筛选
- 按战力排序

### 新增玩家时需要填写
- 数字 ID
- 游戏昵称
- 微信昵称
- 分数
- 战力值
- 擅长位置
- 自我介绍

注意：
- 数字 ID 必须唯一
- 数字 ID 创建后不可修改

## 赛事与报名管理

### 创建赛事
管理员在 `赛事与报名管理` 中可创建赛事，需要填写：
- 赛事名称
- 开始时间
- 每队人数

常见命名示例：
- `3月19日内战`
- `微雨杯 S1`

### 开启或关闭报名
创建赛事后，管理员可以：
- 点击 `开启报名`
- 点击 `关闭报名`

开放报名时，玩家可在首页自行报名。

### 删除赛事
管理员可以删除某个赛事。

注意：
- 删除赛事会一并清理关联的内战会话和拍卖数据
- 删除前建议确认当前赛事不再使用

### 管理员代报名
管理员可在当前赛事中直接选择某位玩家并点击 `代报名`。

适用场景：
- 玩家不在线但需要代为录入
- 线下活动由管理员统一录入报名名单

## 任命队长并初始化赛事
当报名人数基本确定后，管理员可在报名名单中勾选玩家作为队长，并点击：
- `任命队长并初始化赛事`

系统会完成以下动作：
- 保存当前赛事的队长名单
- 初始化内战选人数据
- 让对应队长进入后续选人或拍卖流程

注意：
- 队长必须从已报名玩家中产生
- 如果勾选的人未报名，系统会阻止操作

## 内战选人页使用说明

### 页面布局
内战选人页主要包括：
- 报名面板
- 队长选人区
- 队伍与战力值展示

### 普通玩家
普通玩家在内战选人页可以：
- 查看当前赛事报名情况
- 查看当前队长轮次
- 查看剩余可选选手
- 查看各队当前阵容和总战力

### 队长
队长在轮到自己时可以：
- 在选手池中选择一名选手
- 点击 `确认选择队员`

系统规则：
- 只有当前轮到的队长可以选人
- 非当前队长无法操作
- 选人后系统会实时更新队伍总战力和选人记录

## 选手拍卖页使用说明

### 页面布局
拍卖页分为 3 个区域：
- 左侧：队伍情况
- 中间：当前拍品与竞价区
- 右侧：未拍卖选手与流拍选手列表

### 管理员
管理员可在拍卖配置管理中完成：
- 选择关联赛事
- 设置拍卖标题
- 设置起拍价
- 设置加价幅度
- 设置倒计时秒数
- 为每位队长单独分配预算
- 创建拍卖

创建完成后，管理员可执行：
- `启动拍卖`
- `暂停拍卖`
- `继续拍卖`

### 队长
被任命为队长的玩家在拍卖页中可以：
- 查看自己的队伍预算
- 查看当前拍品
- 输入或调整出价金额
- 点击 `我要出价`

系统规则：
- 出价必须满足当前最低加价要求
- 出价不能超过自己队伍剩余预算
- 倒计时结束后自动成交
- 没有有效出价时，该选手会流拍

## 常见使用流程

### 场景 1：普通内战开赛流程
1. 管理员创建赛事
2. 管理员开启报名
3. 玩家登录并报名
4. 管理员从报名名单中任命队长
5. 管理员初始化赛事
6. 队长进入内战选人页开始选人

### 场景 2：社区赛拍卖流程
1. 管理员创建赛事
2. 玩家报名
3. 管理员任命队长
4. 在拍卖配置管理中为每位队长设置预算
5. 创建拍卖
6. 启动拍卖
7. 队长依次竞价
8. 管理员根据需要暂停或继续拍卖

## 常见问题

### 1. 玩家无法报名怎么办？
请检查：
- 当前赛事是否已开启报名
- 玩家是否已经报名
- 玩家是否使用了正确的数字 ID 登录

### 2. 为什么某个玩家不能被设为队长？
因为队长必须来自当前赛事的报名名单。

### 3. 为什么队长不能选人？
请检查：
- 是否已经初始化赛事
- 当前是否轮到该队长
- 是否已经选中了可选玩家

### 4. 为什么不能出价？
请检查：
- 当前登录账号是否是该拍卖中的队长
- 当前拍卖是否已开始
- 出价是否低于加价要求
- 队伍预算是否足够

## 使用建议
- 管理员在活动开始前，先提前录入玩家库
- 正式开始前，先确认报名名单与队长名单
- 拍卖开始前，先检查每位队长预算是否设置正确
- 删除赛事前先确认是否仍需保留历史数据
